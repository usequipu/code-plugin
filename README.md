# code-plugin

Monaco (VS Code) code editor plugin for [Quipu](https://quipu.cc). Full-featured syntax highlighting and editing for 20+ languages.

## Install

Open the Plugin Manager panel in Quipu → Available → Install.

## Supported file types

`.js`, `.jsx`, `.ts`, `.tsx`, `.py`, `.go`, `.rs`, `.java`, `.c`, `.cpp`, `.css`, `.html`, `.json`, `.yaml`, `.sql`, `.rb`, `.php`, and more.

## Build from source

```bash
npm install
npm run build
```
